# Nanhu High School Visualization Budget
南湖高中預算可視化，為關心學校發展同學提供一個更加方便的方式，以關心地方發展為初始構想，融入資訊技能。

注意:以下作品在Python 3.7.3版本 IPython 7.6.1版本下可完整執行，使用Spyder3編寫。
